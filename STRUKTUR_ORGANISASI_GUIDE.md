## ðŸŽ¬ JAVASCRIPT FUNCTIONS

### `renderBidangSections()`
- Generate 9 bidang section cards
- Populate dengan data dari `memberData` object
- Attach event listeners ke buttons

### `openMemberSwipe(bidangKey)`
- Open modal
- Create member cards for selected bidang
- Create indicator dots
- Attach scroll listener

### `closeMemberSwipe()`
- Close modal
- Restore body scroll

### `updateSwipeIndicator()`
- Detect center card saat scroll
- Update dot indicators
- Update card states (active/inactive)

### `scrollToCard(index)`
- Smooth scroll to specific card
- Used saat click dot indicator

### Event Listeners:
- `Escape` key: Close modal
- `Arrow Left/Right`: Navigate cards
- Carousel scroll: Update indicators
