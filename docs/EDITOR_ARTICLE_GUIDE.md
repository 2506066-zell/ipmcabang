# Panduan Editor Artikel (5 Langkah)

## 1. Isi Informasi Dasar
1. Isi `Judul` minimal 12 karakter.
2. Isi `Penulis`.
3. Pilih `Kategori` dan atur `Tanggal Publish`.

## 2. Tulis Konten Dengan Struktur Aman
1. Gunakan style block `Paragraf`, `Heading 2`, `Heading 3`.
2. Pakai format seperlunya: `Bold`, `Italic`, list, blockquote, link.
3. Hindari format dekoratif berlebih agar hasil tetap konsisten di halaman publik.

## 3. Cek Preview Publish
1. Lihat panel `Preview Publish` untuk memastikan hasil akhir sama dengan halaman artikel publik.
2. Di mobile, gunakan tab `Tulis` dan `Preview`.

## 4. Periksa Panel Kualitas Artikel
1. Pastikan tidak ada error merah.
2. Warning kuning sebaiknya diperbaiki jika memungkinkan.
3. Target minimal konten: 120 kata.

## 5. Publikasikan
1. Klik `Publikasikan` setelah semua valid.
2. Jika ada error kritikal, sistem akan menahan publish sampai diperbaiki.
