Pusat Bantuan

Panduan cepat agar kamu tidak bingung saat menggunakan aplikasi.

Daftar Akun

  1. Buka halaman Daftar.
  2. Isi Nama Panjang sesuai data resmi, pilih Pimpinan, lalu buat Username dan Password.
  3. Tekan Kirim untuk menyelesaikan pendaftaran.

Login

  1. Masukkan Username dan Password.
  2. Centang Ingat Saya jika ingin tetap login.
  3. Tekan Masuk.

Mulai Kuis

  1. Masuk ke menu Quiz.
  2. Pilih set kuis yang tersedia.
  3. Tekan Mulai dan jawab pertanyaan sampai selesai.

Peringkat

  1. Buka halaman Peringkat.
  2. Gunakan pencarian atau filter untuk menemukan nama.
  3. Jika belum ada data, tekan Muat Ulang.

Materi & Artikel

  1. Pilih menu Materi atau Artikel.
  2. Gunakan pencarian untuk topik tertentu.
  3. Tekan item untuk membaca detail.

Instal Aplikasi (PWA)

  1. Buka menu + (FAB).
  2. Pilih Install jika tersedia.
  3. Ikuti petunjuk instalasi di browser.

Masalah Umum

  • Jika halaman lambat, tunggu loader sampai selesai.
  • Jika gagal memuat data, coba refresh halaman.
  • Pastikan koneksi internet stabil.
