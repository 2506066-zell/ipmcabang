Pusat Bantuan & Panduan

Panduan lengkap untuk semua fitur utama agar penggunaan lebih mudah, cepat, dan tidak membingungkan.

Navigasi Cepat

  • Gunakan ikon menu (hamburger) di kanan atas untuk membuka navigasi.
  • Gunakan tombol + (FAB) untuk akses cepat fitur populer.
  • Ikon lonceng di header untuk melihat notifikasi.

Daftar Akun

  1. Buka halaman Daftar.
  2. Isi Nama Lengkap, pilih Asal Pimpinan, lalu buat Username dan Password.
  3. Tekan Kirim untuk menyelesaikan pendaftaran.

Login & Profil

  1. Masukkan Username dan Password.
  2. Masuk, lalu klik ikon profil untuk melihat ringkasan akun.
  3. Gunakan tombol Logout saat selesai.

Notifikasi (In-App & Lock Screen)

  • Notifikasi baru muncul di ikon lonceng.
  • Aktifkan notifikasi agar update muncul di lock screen.
  • Notifikasi otomatis dibersihkan setelah 7 hari, kecuali countdown program kerja.

Program Kerja & Countdown

  • Countdown menampilkan sisa waktu menuju program kerja berikutnya.
  • Countdown juga muncul di modal notifikasi agar mudah dipantau.

Kuis

  1. Masuk ke menu Quiz lalu pilih set kuis yang tersedia.
  2. Jawab semua soal sampai selesai. Timer akan berjalan per soal.
  3. Setiap set kuis hanya bisa dikerjakan 1 kali sampai direset admin.

Catatan: Jika admin mereset set kuis, kamu bisa mengerjakan ulang.

Peringkat

  1. Buka halaman Peringkat untuk melihat ranking.
  2. Nama kamu akan disorot agar mudah ditemukan.

Materi & E-Book

  1. Pilih menu Materi.
  2. Gunakan pencarian atau filter kategori.
  3. Tap materi untuk lihat detail atau unduh file.

Artikel & Berita

  1. Pilih menu Artikel.
  2. Gunakan pencarian dan kategori.
  3. Baca artikel dan gunakan tombol Bagikan atau Salin Tautan.

Instal Aplikasi (PWA)

  1. Buka menu + (FAB) atau tombol install jika tersedia.
  2. Pilih Install.
  3. Ikuti petunjuk dari browser.

Keamanan

  • Jaga kerahasiaan password.
  • Logout jika memakai perangkat bersama.

Masalah Umum

  • Jika data tidak muncul, coba refresh halaman.
  • Jika tombol tidak merespons, pastikan koneksi stabil.
  • Jika notif tidak muncul, aktifkan notifikasi di browser dan perangkat.

FAQ Singkat

  • Kenapa kuis tidak bisa diulang? Setiap set kuis hanya 1x percobaan sampai direset admin.
  • Notif tidak muncul di lock screen? Pastikan izin notifikasi aktif di browser dan perangkat.
  • Kenapa data kosong? Bisa karena belum ada data baru. Coba muat ulang.
  • Bagaimana install aplikasi? Buka menu + (FAB) lalu pilih Install jika tersedia.
