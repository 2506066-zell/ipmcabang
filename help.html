Pusat Bantuan

  • Cara Daftar: buka Halaman Daftar, isi Nama Panjang sesuai data resmi, Pimpinan, Username, Password, lalu kirim.
  • Cara Login: masuk dengan Username dan Password, aktifkan Ingat Saya bila ingin tetap login.
  • Mulai Kuis: nama otomatis dari akun, tekan Mulai; jika nama tidak sesuai database, sistem memberi peringatan.
  • Peringkat: gunakan pencarian dan filter; jika belum ada data, tombol Muat Ulang tersedia.
  • Masalah Umum: jika data lambat, tunggu dengan loader; jika gagal, muat ulang atau coba beberapa saat lagi.
